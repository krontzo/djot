Module djot

Parse and render djot light markup format. See https://djot.net.



parse (input, sourcepos, warn)
Parse a djot text and construct an abstract syntax tree (AST) representing the document. object with pos and message fields.
parse_events (input, warn)
Parses a djot text and returns an iterator over events, consisting of a start position (bytes), and an position (bytes), and an annotation.

object with pos and message fields.
